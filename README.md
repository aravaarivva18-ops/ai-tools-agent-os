# Antigravity AI-Tools CLI

Набор инструментов автоматизации разработки, тестирования и сохранения контекста для ИИ-агентов. Система адаптирована для macOS/Linux и ускоряет цикл разработки ИИ-разработчиков в 3-5 раз за счет встроенной самодиагностики и управления памятью.

---

## ⚙️ Основные функции

- **Авто-лечение ошибок (`agy run`)**: Автоматический запуск тестов, перехват ошибок компиляции/выполнения и генерация точечных исправлений (patches) с помощью LLM без ручного вмешательства.
- **Гибридный поиск по памяти (`agy search`)**: Сверхбыстрый локальный поиск по прошлым хэндоффам и сессиям. Сочетает мгновенный текстовый поиск (время отклика <1 мс) и ленивую загрузку векторной ML-модели FastEmbed для глубокого семантического поиска.
- **Динамическая конфигурация (`agy init`)**: Автоматическое определение корней воркспейсов, исключающее привязку к абсолютным путям.
- **Управление JIT-навыками**: Модуль генерации шаблонов правил и JIT-инструкций (`is_ui`, `is_seo`, `is_scale`, `is_mcp`) под конкретные задачи.

## 🚀 Установка (Быстрый старт)

Для установки системы выполните одну команду в корне проекта:

```bash
./install.sh
```

### Что делает скрипт установки:
1. Синхронизирует зависимости проекта через менеджер пакетов `uv`.
2. Создает базовый конфигурационный файл `.agentic-dev.json` в корне.
3. Проверяет настройки вашего терминала (Zsh/Bash) и регистрирует глобальный псевдоним `agy` для доступа к CLI из любой папки.

После установки обновите настройки терминала:
```bash
source ~/.zshrc   # Для Zsh
# или
source ~/.bashrc  # Для Bash
```

## 💻 Использование CLI `agy`

Доступные консольные команды:

### 1. `agy init`
Инициализирует локальный файл конфигурации `.agentic-dev.json` в текущем проекте.

### 2. `agy run [аргументы pytest]`
Запускает систему автоматического исправления тестов (Self-Healing Loop). Перехватывает выводы падающих тестов, очищает от логов и передает LLM для генерации корректных исправлений в коде.
```bash
agy run
# запуск конкретного теста с флагами
agy run tools/tests/test_config.py -v
```

### 3. `agy search "запрос"`
Быстрый поиск по базе знаний и хандоффам прошлых сессий.
- **Обычный поиск** (по умолчанию): Мгновенно ищет точные текстовые совпадения.
- **Семантический поиск** (с флагом `-s`/`--semantic`): Использует локальную векторную модель для поиска по смыслу.
```bash
agy search "lessons learned"
agy search "ComfyUI API" --semantic
```

## 📐 Интеграция с ИИ (Claude/GPT/Gemini)

Чтобы подключить систему к вашему ИИ-помощнику, скопируйте файлы настроек в корень вашего проекта:
- `CLAUDE.md` — содержит базовые команды CLI и структуру проекта для ИИ.
- `AGENTS.md` — содержит правила разработки (Solo Loop, TDD, YAGNI) для ИИ-агента.

После этого ИИ-агент сможет самостоятельно вызывать `agy run` для починки тестов и использовать `agy search` для восстановления памяти.
