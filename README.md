# 🛠️ AI Tools Workspace (Monorepo)

Централизованное рабочее пространство ИИ-агентов для локального SEO-анализа, маркетинга, квалификации лидов и юридического комплаенса.

---

## 🏗️ Структура проекта

Этот монорепозиторий организован с помощью `uv workspaces` и содержит следующие независимые проекты:

*   **[geo-seo/](file:///Users/rus/ai-tools/geo-seo/)** — парсинг сайтов, SERP-анализ и оценка видимости в AI-поисковиках (GEO).
*   **[ai-sales/](file:///Users/rus/ai-tools/ai-sales/)** — квалификация лидов, скоринг BANT/MEDDIC и генерация PDF-отчетов по продажам.
*   **[ai-marketing/](file:///Users/rus/ai-tools/ai-marketing/)** — анализ конкурентов, генерация контент-планов и маркетинговых отчетов.
*   **[ai-legal/](file:///Users/rus/ai-tools/ai-legal/)** — аудит контрактов, оценка рисков и генерация соглашений.

---

## ⚡ Быстрый старт

Проект использует современный и сверхбыстрый инструмент **`uv`** для управления зависимостями.

### Требования
*   Python >= 3.12
*   Установленный `uv` (`curl -LsSf https://astral.sh/uv/install.sh | sh`)
*   Ollama (для локального оффлайн-инференса)

### Инициализация воркиспейса
Синхронизируйте все зависимости одной командой:
```bash
make sync
```

---

## 🛠️ Команды автоматизации (Makefile)

В корне проекта настроен [Makefile](file:///Users/rus/ai-tools/Makefile) для быстрого выполнения рутинных задач:

*   `make sync` — синхронизация виртуального окружения воркспейса `.venv`.
*   `make check` — запуск линтинга и проверки форматирования (Ruff).
*   `make lint` — автоматическое исправление ошибок качества кода.
*   `make format` — форматирование исходного кода.
*   `make test` — запуск всех автотестов (Pytest).
*   `make clean` — очистка кэшей компилятора Python.

## 🔌 Локальный запуск моделей (Ollama)

Все конфигурации для локального запуска моделей находятся в папке `ollama/`:
*   **Legal Agent:** `ollama create legal-agent -f ollama/Modelfile-legal` (детерминированный, 16k контекст)
*   **Sales Agent:** `ollama create sales-agent -f ollama/Modelfile-sales` (сбалансированный)
*   **Marketing Agent:** `ollama create marketing-agent -f ollama/Modelfile-marketing` (творческий)
