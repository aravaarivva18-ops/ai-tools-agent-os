## 🛠️ 3. Actionable Insights: что адаптировать в `@ai-tools/`

Мы адаптируем ключевые паттерны проектирования из трендов 2026 года без усложнения кодовой базы внешними библиотеками:

1.  **Паттерн Type-Safe Structured Outputs (из Pydantic AI):**
    *   При генерации RAG-ответов или шаблонов кода мы используем стандартную библиотеку `pydantic` (уже установленную в проекте) для жесткой валидации выходных данных ИИ, не подключая тяжелый фреймворк `pydantic-ai`.
2.  **Паттерн Декларативных Инструментов (из Agno):**
    *   Создание ИИ-инструментов как изолированных Python-функций с четкими docstrings и типами. Мы автогенерируем JSON-схему для LLM-функций (как в `rag_index.py`), что минимизирует ручное написание спецификаций.
3.  **Изоляция среды через `uv` (из Browser-Use / Agno):**
    *   Все проекты и утилиты запускаются строго через `uv run`. Это обеспечивает изоляцию, скорость запуска (<100мс для CLI) и отсутствие конфликтов версий.
