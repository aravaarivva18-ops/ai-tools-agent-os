# 💻 Управление ComfyUI через comfy-cli

`comfy-cli` — это официальный интерфейс командной строки для управления установками, окружениями и кастомными нодами в ComfyUI.

---

## ⚡ 1. Быстрый старт (Установка и Запуск)

### Установка CLI
Установка пакета выполняется через `pip` (рекомендуется использовать виртуальное окружение):
```bash
pip install comfy-cli
```

### Настройка автодополнения (опционально)
```bash
comfy --install-completion
```

### Установка ComfyUI
Инициализирует и скачивает чистую версию ComfyUI в текущую директорию:
```bash
comfy install
```

### Запуск сервера
Запускает локальный сервер ComfyUI:
```bash
comfy launch
```
_Сервер по умолчанию запускается на `http://127.0.0.1:8188`._

---

## 🔌 2. Управление кастомными нодами (`comfy node`)

Вы можете искать, устанавливать и обновлять сторонние ноды прямо из терминала без необходимости открывать веб-интерфейс:

### Установка ноды
```bash
comfy node install <node-name>
```

### Обновление всех нод
```bash
comfy node update all
```

### Поиск проблемных нод (Bisect)
Если после установки нод ComfyUI перестал запускаться, можно использовать механизм бисекции для поиска виновного:
```bash
comfy node bisect start
comfy node bisect good  # если запуск успешен
comfy node bisect bad   # если произошел сбой
```

### Создание шаблона новой ноды
Создает структуру и шаблон кода для публикации своей кастомной ноды:
```bash
comfy node init
```

---

## ☁️ 3. Генерация без локального сервера (`comfy generate`)

Инструмент позволяет отправлять prompt на облачные сервера партнеров (Beta-функция):

### Список доступных моделей
```bash
comfy generate list
```

### Запуск генерации через Flux Pro (пример)
```bash
comfy generate flux-pro --prompt "a beautiful cat on the moon, high resolution" --download cat.png
```

---

## 📁 4. Управление воркспейсами (Рабочими путями)

Вы можете переключаться между несколькими инсталляциями ComfyUI на одном компьютере:

### Узнать путь к активному воркспейсу
```bash
comfy which
```

### Установить воркспейс по умолчанию
```bash
comfy set-default /path/to/comfyui/workspace
```

### Запустить команду в определенном воркспейсе
```bash
comfy --workspace=/path/to/another/comfyui launch
```

### Отключение аналитики (Opt-Out)
```bash
comfy tracking disable
```
