## 🛠️ Системные и библиотечные особенности macOS (Дополнения)

* **Локальный TTS на Python 3.14 (macOS):** Стандартная библиотека `kokoro` падает при сборке на Python 3.14 из-за несовместимости spacy/blis. В качестве стабильной и производительной альтернативы для русского языка следует использовать `piper-tts` и модели голосов Nabu Casa с Hugging Face.
* **Использование генераторов в Ruff (RUF015):** При работе с генераторами (например, вывод эмбеддингов в `fastembed`) запрещено преобразовывать генератор в список для взятия первого элемента (`list(gen)[0]`). Используйте конструкцию `next(iter(gen))` для соответствия правилу RUF015.
* **Вызовы Subprocess и Bandit (B603, B607):** При запуске процессов через `subprocess.run` всегда преобразовывайте имя команды в абсолютный путь с помощью `shutil.which("command")`. Для прохождения pre-commit проверок безопасности Bandit добавляйте аннотацию `# nosec B603` в строку вызова.
